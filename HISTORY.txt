# HISTORY

## 2026-06-09

- FastAPI 기반 재고 검색 웹앱 초기 구조를 생성했다.
- 로그인, 세션, 재고 조회 API를 추가했다.
- Google Sheets 연동과 샘플 데이터 fallback을 추가했다.
- HTML/CSS/JavaScript 프론트엔드와 AND/OR 클라이언트 검색을 추가했다.
- `python app/main.py` 직접 실행 시 웹앱 서버가 시작되도록 진입점을 추가했다.
- 재고 조회 화면에 제조사 컬럼과 제조사 버튼 필터를 추가했다.
- 가격 컬럼은 1,000단위 구분 기호로 표시하고 재고수량 0 이하 품목은 숨기도록 변경했다.
- 로컬 아이디/비밀번호 로그인을 Google OAuth 로그인으로 교체했다.
- `data/members.json` 기반 멤버 권한 저장소와 관리자 멤버 관리 API/UI를 추가했다.
- 관리자용 XLSX 재고 파일 업로드 및 Google Sheet 갱신 API/UI를 추가했다.
- Google OAuth 설정 상태 안내와 로컬 전용 개발 로그인 fallback을 추가했다.
- 개발용 로그인 버튼을 기본 표시로 바꾸고 서버 주소 접속 안내를 강화했다.
- 개발용 로그인을 JavaScript fetch 방식에서 서버 리다이렉트 링크 방식으로 변경했다.
- 초기화 흐름을 단일화하고 개발용 로그인 직후 `/?dev=1`에서 재고 화면을 바로 표시하도록 수정했다.
- 브라우저 쿠키 세션이 유지되지 않아도 개발용 로그인 API가 동작하도록 로컬 전용 `X-Dev-Login` 헤더 fallback을 추가했다.
- 정적 파일 캐시를 무력화하고 `/?dev=1` 요청을 서버에서 직접 개발 로그인 처리하도록 변경했다.
- 개발용 로그인 흐름을 제거하고 `/dev`에서 인증 없이 재고 조회 화면을 바로 표시하도록 변경했다.
- `hidden` 속성이 CSS display 규칙에 덮이지 않도록 보정하고 화면 전환 시 display 값을 직접 제어하도록 수정했다.
- Render 배포용 `render.yaml`과 배포 안내 문서를 추가했다.
